Project Idea: Monitoring Land Surface Temperature in Eldoret Town Using Remote Sensing

Project Title:

Monitoring Land Surface Temperature in [Eldoret] Using Remote Sensing"

Objective:

To analyze the changes in Land Surface Temperature (LST) in a rapidly growing urban center over a specified period using remote sensing data.

Study Area:

Eldoret Town

Key Components:

  • Data Collection: Acquire satellite imagery from sources such as Landsat 8, MODIS, or Sentinel-3 for LST analysis. Collect data for different time periods, past 10-15 years to observe trends.
  • Data Preprocessing: Perform atmospheric correction on satellite images. Reproject images to a common coordinate system. Mask out clouds and other noise using QA bands.
  • Land Surface Temperature Retrieval: Use algorithms such as the Split-Window Algorithm (for Landsat) or the Single-Channel Algorithm (for Sentinel-3) to retrieve LST from thermal infrared bands. Validate the LST results using ground-based temperature data, if available.
  • Urban Growth Analysis: Use land cover classification to identify changes in urban areas over time. Map and quantify urban expansion using supervised or unsupervised classification methods.
  • Statistical Analysis: Perform statistical analyses to identify significant trends in LST and UHI over the study period. Use regression analysis to correlate urban expansion with changes in LST.
  • Visualization and Mapping: Create thematic maps showing LST distribution, urban growth, and UHI intensity. Generate time series plots to visualize trends in LST over time.

Tools and Software:

Google Earth Engine for data processing and analysis. QGIS or ArcGIS for spatial analysis and mapping. Python or R for statistical analysis and visualization.

Expected Outcomes:

Detailed maps and time series of LST changes in the urban center.

Potential Applications:

Urban planning and policy-making. Climate change studies and adaptation strategies. Environmental monitoring and sustainable development.
